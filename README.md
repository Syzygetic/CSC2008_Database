# CSC2008_Database
Github Repo for school module CSC2008 Database project.

# How to Run
In Terminal 1, run:

```
cd React\client
node server.js
```

In Terminal 2, run:

```
cd React\client
npm start
```

In Terminal 3, run:
```
cd React\server
npm start
```
